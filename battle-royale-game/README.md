# Battle Royale Tournament Game

A fun Python tournament game where players battle each other using:

- Rock
- Paper
- Scissors
- Fire
- Water

The game automatically creates tournament rounds and announces the final winner.

---

## Features

- Multiplayer Tournament
- Automatic Matchmaking
- Semi-finals & Finals
- Randomized brackets
- Draw replay system

---

## Installation

1. Install Python 3

2. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/battle-royale-game.git
````

3. Open project folder

```bash
cd battle-royale-game
```

4. Run the game

```bash
python main.py
```
